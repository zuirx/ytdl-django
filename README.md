# YTDL web
yt-dlp front-end in django 

## Installation
- Install Python 3.11.*
- Create a venv (and enable it):
```python -m venv venv```
- Install requirements.txt:
```pip install -r requirements.txt```
- Run on windows: run runserver-win.py
- Run on Debian/Linux: run runserver-deb.py
